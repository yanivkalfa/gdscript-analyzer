# Web playground (Phase 5)

A browser playground for gdscript-analyzer — paste GDScript and get diagnostics / completion /
hover entirely client-side, powered by the Rust core compiled to WASM (modeled on Ruff's
`play.ruff.rs` and the Biome playground).

> **Status: Phase 0 placeholder.** Built in Phase 5 once the WASM binding and the Phase-1/2 analysis
> features land. See [plans/PHASE-5-CLIENTS-AND-DISTRIBUTION.md](../plans/PHASE-5-CLIENTS-AND-DISTRIBUTION.md)
> and [plans/research/08-wasm-web-and-bindings.md](../plans/research/08-wasm-web-and-bindings.md).
