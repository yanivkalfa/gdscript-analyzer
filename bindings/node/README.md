# @gdscript-analyzer/core

Node.js bindings for [gdscript-analyzer](https://github.com/yanivkalfa/gdscript-analyzer) — a fast,
embeddable GDScript static-analysis library ("Roslyn for Godot").

> **Status: Phase 0 scaffold.** Not functional yet. The napi-rs binding (built from
> `crates/gdscript-ffi`) is wired in Phase 1, and this package is published to npm from Phase 5.

This directory holds the **npm packaging** for the native addon — the `package.json`, the loader
(`index.js`), the generated type definitions (`index.d.ts`, Phase 1), and the per-platform
`npm/<triple>/` sub-packages wired as `optionalDependencies` (generated by `napi create-npm-dirs`).
It is intentionally **not** a Cargo crate; it packages the `.node` addon produced by the
`gdscript-ffi` crate.

See the repository [README](../../README.md) and
[plans/PHASE-5-CLIENTS-AND-DISTRIBUTION.md](../../plans/PHASE-5-CLIENTS-AND-DISTRIBUTION.md).
